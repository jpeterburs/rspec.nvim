# rspec.nvim

RSpec runner for Neovim. Written in lua.

## Installation

### [lazy.nvim](https://github.com/floke/lazy.nvim)

```lua
{
  'jpeterburs/rspec.nvim',
  cmd = 'RSpec'
}
```

## Usage

### Commands

|Command     |Description|
|------------|------------------------------|
|`RSpec`     |Run rspec on the current file.|
|`RSpecAbort`|Kill running rspec process.   |
